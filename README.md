# GBZ-Ref-Suite
This is the one size fits all tool for referees,  containing your basic stopwatch feature, alongside others such as match logs, choosing who tags first, etc.

GBZ Ref Suite Tutorial

Installation:
### APK
**Phone**
- Download the apk file
- Open the file and extract it

**VR Headset**
- Download the apk file onto your computer
- Download sidequest (advanced installer) from https://sidequestvr.com/setup-howto
- Log into sidequest
- Connect your headset (make sure debeloper mode is enabled, if not you can look here to do it https://knowledge.vr-expert.com/kb/how-to-activate-developer-mode-on-the-meta-quest-2-or-quest-pro/)
- Drag and drop the apk and it will install to the unknown sources tab

### HTML
- Download the file
- Open the file, this will open a browser tab that contains the Ref Suite

Do note that the APK version has some strange scaling issues compared to the HTML file
